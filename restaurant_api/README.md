Restaurant Api
==============
## Packages Required

* **Ruby2.3.0p0 (2015-12-25 revision 53290) [x86_64-linux]**

* **Rails 4.0.0**

  Installed when you bundle.

## Installation

* **Clone project**
  ```bash
  git clone git@github.com:webonise/NFHS-ROR.git
  ```

* **Install gems**
  ```bash
  bundle install
  ```

* **checking json response**
  ```bash
  restaurants.json
  ```
